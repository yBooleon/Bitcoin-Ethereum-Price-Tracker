README.md# btc/eth - crypto price tracker

Simple Bitcoin & Ethereum real-time price tracker.

## 🚀 About this project
This is my **first web development project**, created while taking my very first steps in learning HTML, CSS, and JavaScript. The goal was to understand how to fetch data from an external API (CoinGecko) and build a functional user interface. I'm excited about what I've learned so far!

## Features
- Real-time price tracking for **Bitcoin (BTC)**.
- Real-time price tracking for **Ethereum (ETH)**.
- Clean and responsive user interface.

## How to use
1. Open the `index.html` file in your web browser.
2. Click on the buttons to fetch the latest price in USD.

## Technologies used
- HTML5
- CSS3 (Styling)
- JavaScript (API consumption)
